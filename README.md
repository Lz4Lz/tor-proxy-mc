# Tor-proxy

This mod allows you to route your traffic through Tor and also join .onion servers.

## Requirements

* Minecraft Java Edition 1.21.11+
* Fabric API
* Mod Menu (Optional, it provides a way to configure the mod)
* Tor Service (For the proxy setup on your system)

## Configuration

Use the in-game Mod Menu and double click the mod "tor-proxy". You can enable/disable the proxy and change host:port here.
